XI. DUAL CREAM DOSER IN PVC PACKAGING

MACHINE ELEMENTS

TECHNICAL DESCRIPTION

The cream doser comprises two 500g cylinders for dosing into one container, with a total dosing capacity of 2×500g, representing 1kg in one cycle. The drive for the pistons is electromotor, and weight adjustment is automatic, achieving an accuracy of less than 1%.

The movable belt automatically places the containers using the loader, which positions them for filling. Then they move onto the vibrating table and the presentation area. When the container reaches the filling position, a sensor detects it and automatically initiates the dosing. Following the dosing, a lid is placed on the containers and automatically sealed.

TECHNICAL SPECIFICATIONS

  • Capacity: Maximum 600 kg/h
  • Containers 200-300g: Approximately 750 units/hour = 150-225 kg/h
  • Containers 400-500g: Approximately 700 units/hour = 280-350 kg/h
  • Containers 800-1000g: Approximately 600 units/hour = 480-600 kg/h
  • The entire system of the doser is made from stainless steel, prochrome
  • Doser dimensions: length 4000 mm, width 700 mm, height 1500 mm
  • Automatic jar placement on the belt from the loader
  • Tools for lid closure
  • Automatic jar guiding
  • Automatic lid sealing
  • Process control system
